## Installation
To set up and run the project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/Ryder.git
   cd cses-web-dev
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
